GHID RAPID · OPTIMIZAT PENTRU MOBIL

Configurează și înregistrează corect fiecare cursă

Urmează pașii în ordine. Înainte de START, completează datele din Settings și acceptă toate permisiunile cerute.

Începe ghidul
1SettingsCompletează datele
2PermisiuniAcceptă accesul
3START / STOPÎnregistrează cursa
4Save + SaveSalvează de două ori
5ArchiveArhivează sau exportă
Pasul 1 din 11
Pasul 1

Pornește de pe pagina principală

Aceasta este pagina de început. Nu apăsa încă START. Mai întâi intră în tab-ul Settings din bara de jos.

Acțiune: apasă pictograma Settings din colțul dreapta-jos.
Pasul 2

Completează datele obligatorii

Datele introduse aici vor apărea în fișele zilnice, în arhivă și în exportul Excel.

  • Active Event Name — numele evenimentului
  • Car Plate Number — numărul de înmatriculare
  • Driver Name — numele șoferului
  • Vehicle Type — tipul vehiculului
  • Fuel Type — tipul combustibilului
Opțional: păstrează activ „Display city name” pentru afișarea localității.
Pasul 3

Revino la pagina principală și apasă START

După completarea setărilor, întoarce-te în tab-ul Sheets.

Acțiune: apasă START.
Important: la prima utilizare vor apărea mai multe permisiuni Android. Acceptă-le în ordinea în care apar.
Pasul 4

Acceptă accesul la locație în fundal

Permisiunea permite continuarea măsurării și când aplicația nu este deschisă pe ecran.

Acțiune: în fereastra „Background Location Access”, apasă Accept.
Pasul 5

Permite notificările

Notificările indică faptul că aplicația rulează și monitorizează cursa.

Acțiune: apasă Allow.
Pasul 6

Permite locația precisă

Păstrează selectată opțiunea Precise și permite accesul inițial la locație.

Acțiune: selectează While using the app.
Pasul 7

Activează „Allow all the time”

Aceasta este setarea esențială pentru monitorizarea traseului în fundal.

Acțiune: selectează Allow all the time și păstrează activ Use precise location.
Dacă pagina nu se deschide automat:
Settings telefon → Apps → Qelvi → Permissions → Location → Allow all the time.
Pasul 8

Folosește START, STOP și meniul cu trei puncte

După acordarea permisiunilor, timpul și distanța încep să se actualizeze.

STARTPornește o cursă nouă.
STOPÎncheie cursa și salvează traseul.
⋮Deschide opțiunile evenimentului.
Important: la final folosește întotdeauna butonul STOP.
Pasul 9

Gestionează ziua și evenimentul

După STOP, ziua apare ca o fișă cu data, vehiculul, șoferul și totalul de kilometri.

  • Add ManualAdaugă manual o cursă sau o distanță.
  • Archive EventMută întregul eveniment în arhivă.
  • Delete EventȘterge evenimentul și datele sale.
Pentru editare: deschide fișa zilei sau apasă pictograma creion.
Pasul 10

Editează traseele și salvează de două ori

Poți modifica data, datele generale, locurile, orele și numărul de kilometri.

  1. Deschide Trip Routes.
  2. Editează cursa sau apasă Add row.
  3. Apasă Save în pagina „Trip rows”.
  4. Revino și apasă din nou Save în pagina „Edit day sheet”.
Foarte important: fără al doilea Save, modificările pot să nu fie păstrate.
Pasul 11

Arhivează, restaurează sau exportă

Evenimentele arhivate se găsesc în tab-ul Archive.

  • Pictograma verdeRestaurează evenimentul și îl face activ.
  • Coșul roșuȘterge definitiv evenimentul.
  • DescărcareExportă datele evenimentelor arhivate.
Pentru evenimentul activ poți folosi și „Export Active Event to Excel”.
✓

Configurarea este finalizată

Verifică Settings, pornește cu START, oprește cu STOP și salvează toate modificările.

REZUMAT

Procesul complet

  1. Intră în Settings și completează datele.
  2. Revino în Sheets și apasă START.
  3. Acceptă accesul în fundal, notificările și locația.
  4. Activează „Allow all the time”.
  5. Oprește cursa cu STOP.
  6. Editează traseele și apasă Save de două ori.
  7. Arhivează, restaurează sau exportă evenimentul.
